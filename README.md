# DeltaLake: 
-Permite actualizar, insertar o eliminar valores de un archivo, registrando la fecha y hora en la que se realizó la modificación
-El uso se divide en 3 pasos:
  Bronze Layer -> Silver Layer -> Gold Layer

  Bronze: Ingreso de los datos sin modificaciones
  Silver: Datos limpios y filtrados
  Gold: Data lista para usarse
